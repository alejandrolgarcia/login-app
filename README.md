# LoginApp

_Este proyecto tiene como objetivo trabajar con un proceso de autenticación por token tradicional._

## Conocimientos aplicados

* Tokens
* Validar formularios
* LocalStorage
* Borrar tokens del localStorage
* Creacion de usuarios
* Firebase REST API
